# 📝 Daily Journal

A simple, secure daily journaling web application that allows users to write, view, and manage their personal journal entries. Built with Node.js, Express, and EJS on the backend, and Passport.js for user authentication.

🌐 **Live Demo**: [https://dailyjournal-4fmz.onrender.com](https://dailyjournal-4fmz.onrender.com)

---

## ✨ Features

- 🖊️ Add and delete journal entries
- 🔐 User authentication using Passport.js
- 📅 Date-based journal organization
- 🎨 Clean and responsive UI with HTML, CSS, and EJS
- 🔒 Secure session management with `express-session`

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript
- EJS templating

### Backend
- Node.js
- Express.js
- Passport.js (Local Strategy)
- express-session
- body-parser
- mongoose (for MongoDB integration)

## 🚀 Getting Started

### Prerequisites
- Node.js and npm
- Postgre.sql (reffer to db.sql folder)
- setup the postgresql before starting project installation you will see all the sql queries used to create table in db.sql file

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sudarshan2404/DailyJournal.git
   ```
   ```bash
   cd daily-journal
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start the server**
   ```bash
   node app.js
   ```
4. **Open your browser and navigate to http://localhost:3000**
